# 章執筆

指定された章の本文を執筆せよ。

## 入力

- プロジェクトID: $ARGUMENTS[0]
- 章番号: $ARGUMENTS[1]
- `outputs/<project-id>/worldview.md`（世界観）
- `outputs/<project-id>/characters.md`（キャラクター）
- `outputs/<project-id>/plot.md`（プロット骨子）

---

## 世界観・設定用語の扱い方（最重要）

**設定は「見せびらかす」ものではない。物語に溶け込ませる。**

### 原則

1. **専門用語は最小限に**
   - 用語を使わなくても伝わるなら使わない
   - 1章に新規用語は2〜3個まで

2. **初出時は自然に補足**
   - 括弧書きの説明はNG
   - 地の文や行動で意味が伝わるようにする

3. **連打禁止**
   - 同じ用語を短い間隔で繰り返さない
   - 「調律」「調律士」「調律杖」と並べない

4. **あえて説明しない勇気**
   - 読者が想像で補える部分は説明不要
   - 雰囲気で伝わるなら雰囲気で

### 悪い例 → 良い例

```
❌ 調律士として十年以上生きてきた俺には、相手の振動を感じ取る癖がある。
   普通の人間なら、心臓の鼓動、血流のリズム、呼吸の波。
   それらが混ざり合った「生命の振動」が伝わってくる。

✅ 人は誰でも、生きているだけで空気を揺らす。
   心臓が動き、血が巡り、息をするたびに。
   だが目の前の少女からは——何も感じなかった。
```

```
❌ 第六オクターブの調律士だった俺は、楽団「残響」のリーダーとして
   大競演会に出場し、調律をミスしてマルクの振動を暴走させた。

✅ 三年前、俺のせいでマルクは腕を失った。
   何千人の前で、たった一瞬の狂いが——
   あの日から、俺は音を捨てた。
```

```
❌ 「ここは調律禁止だ。看板も読めないのか」

✅ 「ここじゃ魔法は禁止だ。看板も読めないのか」
   （※「調律」という用語を使わなくても伝わる場面では一般語で）
```

### 用語導入のタイミング

| 用語の重要度 | 導入方法 |
|:--|:--|
| 物語の核心 | 自然な文脈で丁寧に導入 |
| 世界観の装飾 | サラッと出して流す |
| なくても困らない | 使わない |

## 執筆ガイドライン

### 文体
- 一人称視点
- 軽快なテンポ
- 会話文多め（7:3で会話:地の文）

### 構成（1章あたり）
- 目安: 3000〜5000文字
- シーン数: 2〜3
- 章末に次章への引きを入れる

### 表現
- 五感を使った描写
- キャラの感情をモノローグで表現
- 過度な説明は避ける（見せる、語らない）
- **設定説明ではなく、行動と感情で物語を進める**

---

## 出力

`outputs/<project-id>/chapters/XX.md` に出力（XXは章番号）:

```markdown
# 第X章「章タイトル」

（本文）
```

## 注意点

- プロット骨子の該当シーンを忠実に展開
- キャラクターの口調・性格を一貫させる
- 世界観設定との矛盾を避ける
- **設定資料を読んだからといって、それを全部出そうとしない**
- **読者は設定を読みたいのではなく、物語を読みたい**
