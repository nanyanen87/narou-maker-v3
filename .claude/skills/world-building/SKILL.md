# 世界観構築

3つの単語を受け取り、それらを織り込んだファンタジー世界を構築せよ。

## 入力

- プロジェクトID: $ARGUMENTS[0]
- 単語1: $ARGUMENTS[1]
- 単語2: $ARGUMENTS[2]
- 単語3: $ARGUMENTS[3]

## ベース

ファンタジー世界（剣と魔法）を基盤とする。

## キーワードの活用方法

各単語は以下のいずれかの方法で世界観に組み込む:

| 活用タイプ | 説明 | 例（単語: 時計） |
|:--|:--|:--|
| **具体的** | そのまま存在として登場 | 時計塔が街の中心にある |
| **比喩的** | 概念・構造のメタファー | 時間に支配された階級社会 |
| **フレーバー** | 雰囲気・文化的装飾 | 「刻を刻む」という慣用句がある |

3単語すべてを使う必要があるが、配分は自由。
1つを主軸に、残りをサブ要素にしてもよい。

`outputs/<project-id>/worldview.md` に出力:

## 人間への確認

以下の場合は `AskUserQuestion` を使って人間に判断を委ねる:

- **単語の解釈が複数ある場合**
- **世界観の方向性を決める場合**
- **活用タイプの確認**
- **重大な設定の分岐**

**選択肢は必ず番号付きで提示する:**

```
「影」の解釈を選んでください:
1. 闇属性魔法の体系
2. 暗殺者ギルドの存在
3. 影の国という異界
4. その他（自由入力）
```

自分で判断できる場合は進めてよい。迷ったら聞く。

## 注意点

- 3単語がフレームワークの各層に影響を与えるよう設計する
- 全項目を埋める必要はない。3単語と関連が深い項目を重点的に
- 項目間の整合性を保つ（例: 死の定義と宗教は矛盾しない）
- 後続工程（キャラ・プロット）の土台となる「世界のルール」を明確に
