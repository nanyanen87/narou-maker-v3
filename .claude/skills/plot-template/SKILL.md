# プロット構築

世界観設定を元に、物語のプロットを作成せよ。

---

## 最重要：短編の面白さとは何か

**短編は「長編を短くしたもの」ではない。短いからこそできる芸がある。**

構造テンプレートを埋める前に、**面白さの核**を決めろ。

### 面白さの5要素

| 要素 | 説明 | 例 |
|:--|:--|:--|
| **意外性** | 読者の予想を裏切る | O・ヘンリー「賢者の贈り物」：互いの宝物を売って贈り物を買う皮肉 |
| **切り口** | 当たり前を別角度で見せる | 芥川「羅生門」：善悪の境界を問う |
| **凝縮** | 一瞬に人生を詰め込む | ヘミングウェイ「老人と海」：一回の漁に人生を凝縮 |
| **余韻** | 読後に残るもの | カーヴァー：語られないことで語る |
| **声** | 独特の語り口 | 村上春樹：あの文体でしか成立しない世界 |

### やってはいけないこと

```
❌ テンプレート通りの予定調和
❌ 設定を消化するだけの物語
❌ 驚きのない展開
❌ キャラが役割を演じてるだけ
```

---

## ステップ1：面白さの核を決める（最初にやる）

**構造より先に「何が面白いのか」を一文で言え。**

```
この物語の面白さ：
「○○だと思っていたら、実は○○だった」（意外性）
「○○という視点で○○を描く」（切り口）
「○○の一瞬に○○が詰まっている」（凝縮）
```

### 人間への確認

`AskUserQuestion` で核を決める：

```
この物語の「面白さの核」は何にしますか？
1. 意外性（読者の予想を裏切るオチ）
2. 切り口（普通と違う視点・解釈）
3. 凝縮（一瞬に人生を詰め込む）
4. 余韻（語らないことで語る）
5. その他（自由入力）
```

核が決まったら、それを実現する具体的なアイデアを確認：

```
具体的にどんな意外性/切り口/凝縮にしますか？
（例：「助けに来たと思った仲間が実は黒幕」「敵視点で主人公を描く」等）
```

---

## ステップ2：構造を選ぶ（核を活かすために）

核が決まったら、それを**活かす構造**を選ぶ。構造が主役じゃない。核が主役。

### テンプレート一覧

| 型 | 構造 | 相性の良い核 |
|:--|:--|:--|
| A: シンデレラ型 | 不遇→転機→上昇→危機→逆転 | 意外性（実は○○だった） |
| B: 行きて帰りし | 日常→異界→試練→宝→帰還 | 凝縮（旅に人生を詰める） |
| C: 悲劇型 | 上昇→頂点→過ち→転落→破滅 | 切り口（善悪の境界） |
| D: 再生型 | 囚われ→苦悩→光→変化→解放 | 余韻（変化の予感で終わる） |

**短編なら3ビートで十分。5ビートは中編向け。**

---

## ステップ3：キャラクターを決める

キャラは「面白さの核」を実現するための駒。

```
この核を成立させるために必要な人物は誰か？
- 意外性のオチを担う人物
- 切り口を体現する人物
- 読者の感情移入先
```

類型的なキャラを避ける。**そのキャラにしかできないこと**を考える。

---

## 注意点

- **テンプレートを埋めることがゴールじゃない**
- 面白さの核がないプロットは却下
- 予定調和を避ける
- 読者に「次どうなる？」と思わせる
- 世界観設定は活かすが、設定を消化するための物語にしない
