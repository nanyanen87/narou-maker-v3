# プロット構築

世界観設定を元に、物語のプロットを作成せよ。

---

## 最重要：短編の面白さとは何か

**短編は「長編を短くしたもの」ではない。短いからこそできる芸がある。**

構造テンプレートを埋める前に、**面白さの核**を決めろ。

### 面白さの5要素

| 要素 | 説明 | 例 |
|:--|:--|:--|
| **意外性** | 読者の予想を裏切る | O・ヘンリー「賢者の贈り物」：互いの宝物を売って贈り物を買う皮肉 |
| **切り口** | 当たり前を別角度で見せる | 芥川「羅生門」：善悪の境界を問う |
| **凝縮** | 一瞬に人生を詰め込む | ヘミングウェイ「老人と海」：一回の漁に人生を凝縮 |
| **余韻** | 読後に残るもの | カーヴァー：語られないことで語る |
| **声** | 独特の語り口 | 村上春樹：あの文体でしか成立しない世界 |

### やってはいけないこと

```
❌ テンプレート通りの予定調和
❌ 設定を消化するだけの物語
❌ 驚きのない展開
❌ キャラが役割を演じてるだけ
❌ 安全で古典的な案で妥協する
```

---

## 重要：「トーン」と「核」は独立している

| 概念 | 説明 | 例 |
|:--|:--|:--|
| **トーン** | 文体、雰囲気、読み心地 | ほのぼの、シリアス、コメディ |
| **核** | 物語の構造、オチ、仕掛け | 意外性、切り口、凝縮 |

**この2つは矛盾しない。**

- 「ほのぼの日常系」（トーン）× 「世界観がひっくり返る」（意外性）= 成立する
- 「シリアス」（トーン）× 「余韻で語る」（余韻）= 成立する

トーンに引っ張られて核を弱めるな。ほのぼの日常系だからといって、意外性を控えめにする必要はない。

---

## 短編を面白くする5つの技術

**核が決まったら、以下の技術でそれを活かす。**

### 1. 入口が強い（開始10秒で勝つ）

短編は助走がない。最初の数行で勝負が決まる。

**やれ：**
- 異常な状況から始める（死体の隣で目覚める、法律が変わった世界）
- いきなり価値観にパンチ（「この村では嘘をつくと罰金」）
- 主人公が既に行動している状態から始める

**やるな：**
- 風景描写から始める
- 「ある日」「昔々」
- キャラ紹介から始める

### 2. 情報の出し方が上手い（読者の脳を働かせる）

短編の快感は「理解が追いついた瞬間」に出る。

**やれ：**
- 説明しないで見せる（ルールを会話や行動で悟らせる）
- 伏線を"目立たない形"で置く（読み返すと見える）
- 読者に「あ、そういうことか」と思わせる

**やるな：**
- 地の文で世界観を説明
- 「実は〜だったのだ」という種明かし
- 全部言葉で説明する

### 3. ひっくり返しがある（でもズルくない）

オチは「驚かせた」より「腑に落ちた」が大事。

**強いひっくり返し：**
- 事実の反転：状況の解釈が変わる
- 視点の反転：語り手が信用できない／立場が逆だった
- 価値の反転：良いと思っていたものが悪、またはその逆

**弱いひっくり返し（避けろ）：**
- 「実は夢でした」
- 「実は死んでいました」
- 後出しの新情報で解決

### 4. 余白がある（読者に最後の仕事を残す）

短編は全部を言い切らない方が強いことが多い。

**やれ：**
- 最後の一行で景色が変わる
- 読者の中で「続きが勝手に生成される」終わり方
- 言わないことで言う

**やるな：**
- エピローグで全部説明
- 「こうして彼は幸せになりました」
- 読者の解釈の余地を奪う

### 5. 核は一個だけ（テーマが絞れてる）

短編は欲張ると死ぬ。

```
「この話で刺したい感情はこれ」って一本に絞る。
喪失、罪悪感、救い、滑稽さ、怖さ、恋、尊厳…のどれか一発。
```

**チェックリスト：**
- [ ] この話の核を一文で言える
- [ ] その核以外の要素は削れる
- [ ] 全シーンが核に貢献している

## ステップ1：面白さの核を決める（最初にやる）

**構造より先に「何が面白いのか」を一文で言え。**

```
この物語の面白さ：
「○○だと思っていたら、実は○○だった」（意外性）
「○○という視点で○○を描く」（切り口）
「○○の一瞬に○○が詰まっている」（凝縮）
```

### 人間への確認

`AskUserQuestion` で核を決める：

```
この物語の「面白さの核」は何にしますか？
1. 意外性（読者の予想を裏切るオチ）
2. 切り口（普通と違う視点・解釈）
3. 凝縮（一瞬に人生を詰め込む）
4. 余韻（語らないことで語る）
5. その他（自由入力）
```

核が決まったら、それを実現する具体的なアイデアを**極端に**提案する。

### 意外性の場合：極端に振れ

**絶対にやるな：**
- 「実は黒幕だった」レベルの古典的オチ
- 「死んでいた」「夢だった」のような使い古されたオチ
- トーンに配慮した控えめな意外性

**やれ：**
- 世界の前提がひっくり返る（例：主人公が人間じゃなかった、舞台が地球じゃなかった）
- 読者が読んでいた物語自体がフィクション内フィクションだった（メタ構造）
- 時間・空間・因果の常識が崩れる
- 「そんなのありかよ」と思わせるレベル

**極端な意外性の例：**
```
- 主人公は自分を人間だと思っているが、実は世界が生成した自動装置だった
- 物語の舞台だと思っていた場所が、実は別の何かの内部だった
- 読者が読んでいるこのテキスト自体が、作中世界で発掘された遺物だった
- 登場人物全員が同一人物の別人格だった
- 時系列が逆だった（最初のシーンが実は最後だった）
```

5つ以上の極端な案を出し、人間に選ばせる。安全な案は出すな。

---

## ステップ2：構造を選ぶ（核を活かすために）

核が決まったら、それを**活かす構造**を選ぶ。構造が主役じゃない。核が主役。

### テンプレート一覧

| 型 | 構造 | 相性の良い核 |
|:--|:--|:--|
| A: シンデレラ型 | 不遇→転機→上昇→危機→逆転 | 意外性（実は○○だった） |
| B: 行きて帰りし | 日常→異界→試練→宝→帰還 | 凝縮（旅に人生を詰める） |
| C: 悲劇型 | 上昇→頂点→過ち→転落→破滅 | 切り口（善悪の境界） |
| D: 再生型 | 囚われ→苦悩→光→変化→解放 | 余韻（変化の予感で終わる） |

**短編なら3ビートで十分。5ビートは中編向け。**

---

## ステップ3：キャラクターを決める

キャラは「面白さの核」を実現するための駒。

```
この核を成立させるために必要な人物は誰か？
- 意外性のオチを担う人物
- 切り口を体現する人物
- 読者の感情移入先
```

類型的なキャラを避ける。**そのキャラにしかできないこと**を考える。

---

## 注意点

- **テンプレートを埋めることがゴールじゃない**
- 面白さの核がないプロットは却下
- 予定調和を避ける
- 読者に「次どうなる？」と思わせる
- 世界観設定は活かすが、設定を消化するための物語にしない
