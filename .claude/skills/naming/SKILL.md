# 命名スキル

世界観設定（worldview.md）を参照し、統一感のある名前を生成する。
ファンタジー、SF、現代、歴史など全ジャンルに対応。

## 入力

- プロジェクトID: $ARGUMENTS[0]
- カテゴリ: $ARGUMENTS[1]（person/place/item/organization/ability/creature）
- サブカテゴリ: $ARGUMENTS[2]（任意）
- 生成数: $ARGUMENTS[3]（デフォルト: 5）

## 前提条件

`outputs/<project-id>/worldview.md` が存在すること。
存在しない場合は `/world-building` の実行を促す。

---

## ジャンル別の命名特性

| ジャンル | 特徴 | 言語傾向 |
|:--|:--|:--|
| **ファンタジー** | 架空言語、神話的響き | 古語、造語、神話由来 |
| **SF** | 技術的、コードネーム風 | 略語、数字混合、人工言語 |
| **現代** | リアルな命名規則 | 実在言語、流行名 |
| **歴史** | 時代・地域に忠実 | 当時の命名慣習 |
| **和風** | 漢字の意味、音の響き | 日本語、古語 |
| **中華風** | 姓+名、字号、二つ名 | 中国語音韻 |
| **異世界転生** | 現代名+異世界名の混在 | ハイブリッド |

## カテゴリ体系

### 1. 人名（person）

| サブカテゴリ | 説明 | ジャンル別例 |
|:--|:--|:--|
| **commoner** | 一般市民・平民 | ファンタジー: 短く素朴 / 現代: 普通の名前 / SF: ID風 |
| **elite** | 上流階級・貴族・幹部 | ファンタジー: 荘厳 / 現代: 旧家風 / SF: コードネーム |
| **protagonist** | 主人公・英雄 | 覚えやすく印象的 |
| **professional** | 専門職（学者・医師・技術者等） | 職業を反映、敬称あり |
| **merchant** | 商人・実業家 | 実直、信頼感 |
| **religious** | 宗教者・精神的指導者 | 神聖、教義由来 |
| **antagonist** | 敵役・ライバル | 不穏さ、対比を意識 |

### 2. 地名（place）

| サブカテゴリ | 説明 | ジャンル別例 |
|:--|:--|:--|
| **nation** | 国・大規模勢力圏 | 歴史: 実在参照 / SF: 惑星名・コロニー名 |
| **city** | 都市・主要拠点 | 現代: 実在都市風 / SF: ステーション名 |
| **settlement** | 村・集落・居住区 | 地形・産業由来 |
| **terrain** | 自然地形（山/川/森/海等） | 現象・伝説由来 |
| **facility** | 施設・ダンジョン・基地 | 用途・コード名 |
| **building** | 建物・店舗 | ブランド名・看板名 |

### 3. アイテム（item）

| サブカテゴリ | 説明 | ジャンル別例 |
|:--|:--|:--|
| **weapon** | 武器・兵器 | ファンタジー: 伝説風 / SF: 型番 / 現代: ブランド名 |
| **protection** | 防具・防護装備 | 守護を連想、機能名 |
| **tool** | 道具・機器・アーティファクト | 効果・用途を示す |
| **material** | 素材・資源 | 特性を反映 |
| **consumable** | 消耗品（薬・弾薬等） | 効果・規格を示唆 |
| **legendary** | 伝説級・ユニーク品 | 固有名詞、来歴あり |

### 4. 組織（organization）

| サブカテゴリ | 説明 | ジャンル別例 |
|:--|:--|:--|
| **government** | 政府・国家機関 | 権威・正統性を示す |
| **guild** | ギルド・組合・協会 | 象徴的なシンボル |
| **military** | 軍事組織・騎士団・部隊 | 誇り高い、動物・数字由来 |
| **religion** | 宗教組織・教団・宗派 | 神聖、教義由来 |
| **underground** | 秘密結社・犯罪組織 | 暗示的、隠語 |
| **corporate** | 企業・商会・財閥 | 繁栄、創業者名、略称 |

### 5. 能力・システム（ability）

| サブカテゴリ | 説明 | ジャンル別例 |
|:--|:--|:--|
| **skill** | スキル・技能 | 効果を端的に |
| **power** | 魔法・超能力・異能 | 属性+効果、詠唱風、科学用語風 |
| **title** | 称号・二つ名 | 功績・特性を反映 |
| **rank** | ランク・階級・レベル | 序列を示す（数字、記号、色等） |

### 6. 種族・生物（creature）

| サブカテゴリ | 説明 | ジャンル別例 |
|:--|:--|:--|
| **race** | 種族・民族・異星人 | 文化・特徴を反映 |
| **enemy** | 敵性生物・モンスター | 脅威度を示唆 |
| **mythical** | 幻獣・神獣・伝説生物 | 神話的、荘厳 |
| **companion** | 使い魔・ペット・AI | 親しみやすい〜神秘的 |

## 言語的アプローチ

### 音韻パターン（フォネティクス）

世界観・ジャンルに応じた音の特徴を定義し、一貫性を保つ。

| 言語ベース | 特徴 | 適したジャンル |
|:--|:--|:--|
| **ラテン系** | 流麗、母音終わり（-us, -a, -um） | ファンタジー、歴史（ローマ） |
| **ゲルマン系** | 硬質、子音重視（-heim, -burg） | ダークファンタジー、北欧風 |
| **スラブ系** | 力強い（-ov, -ev, -ski） | 東欧風、冷戦スパイ物 |
| **ケルト系** | 神秘的（-wyn, -ael, -ith） | 妖精、神秘ファンタジー |
| **和風** | やわらか、漢字の意味 | 和風ファンタジー、時代劇 |
| **中華風** | 音節明確、字の意味 | 中華ファンタジー、武侠 |
| **英語系** | 現代的、国際的 | 現代物、ライトノベル |
| **人工言語風** | 造語、数字混合 | SF、サイバーパンク |

### 語尾・語形による印象操作

| パターン | 印象 | 例 |
|:--|:--|:--|
| **-a, -ia, -ina** | 女性的、柔らか | Aria, Lumina, 美奈 |
| **-us, -os, -on** | 男性的、古典的 | Marcus, Leon |
| **-el, -iel** | 天使的、高貴 | Gabriel, Uriel |
| **数字・記号混合** | 未来的、人工的 | Unit-07, K-9, A.L.I.C.E. |
| **略称・頭字語** | 組織的、技術的 | S.H.I.E.L.D., NERV |
| **漢字二字** | 東洋的、意味深 | 白夜、紅蓮、天命 |

### 意味を込める技法

| 技法 | 説明 | 適用例 |
|:--|:--|:--|
| **花言葉** | 花の名前・意味を活用 | Dahlia（威厳）, 桜（儚さ） |
| **古語借用** | 死語・古典語から | Lux（光）, 月讀（ツクヨミ） |
| **アナグラム** | 単語の並べ替え | heart → Thera |
| **合成語** | 意味のある語を組み合わせ | Ironforge, 鋼鉄城 |
| **神話引用** | 神話の名前をアレンジ | Thor → Thorin, 天照 → アマテル |
| **音訳** | 外国語の音を借用 | Galaxy → ギャラクシア |
| **型番風** | 製品・兵器らしさ | RX-78, Type-00 |

## 出力

### 命名リスト

生成した名前は以下の形式で返答し、必要に応じて `outputs/<project-id>/names.md` に追記する。

```markdown
## [カテゴリ] - [サブカテゴリ]

| 名前 | 読み | 意味・由来 | 備考 |
|:--|:--|:--|:--|
| Aldrich | アルドリッチ | 古（ald）+ 支配者（ric） | 貴族向け |
| Elara | エラーラ | ギリシャ神話の月の名 | 女性名 |
| ... | ... | ... | ... |
```

### 命名規則書（初回生成時）

初めてこのスキルを使用する際は、世界観に基づいた「命名規則書」を作成する。

`outputs/<project-id>/naming-rules.md`:

```markdown
# 命名規則書

## 作品情報

- ジャンル: [ファンタジー/SF/現代/歴史/和風/etc.]
- 舞台: [異世界/未来/現代日本/etc.]
- 雰囲気: [シリアス/コメディ/ダーク/etc.]

## この世界の言語的特徴

### 基本音韻
- 主要言語ベース: [ラテン系/ゲルマン系/和風/etc.]
- 好む音: [例: 流麗な母音、硬質な子音]
- 避ける音: [例: 発音しにくい子音連続]

### 地域・勢力別の傾向

| 地域/勢力/種族 | 命名傾向 | 語尾パターン | 例 |
|:--|:--|:--|:--|
| ... | ... | ... | ... |

### 身分・階級による違い

| 身分/階級 | 名前の長さ | 付加要素 | 特徴 |
|:--|:--|:--|:--|
| 一般市民 | 短い | なし | シンプル |
| 上流階級 | 長い | 称号・家名 | 格式を示す |

### アイテム・組織の命名規則

| カテゴリ | 命名スタイル | 例 |
|:--|:--|:--|
| 武器 | [伝説風/型番/etc.] | ... |
| 組織 | [略称/正式名/etc.] | ... |

### 禁忌・避けるべき名前

- [既存作品との被り]
- [文化的タブー]
- [読者が混乱する類似名]
```

## 人間への確認

以下の場合は `AskUserQuestion` を使って人間に判断を委ねる:

- **言語ベースの選択**（世界観から明確でない場合）
- **文化圏の設定**（複数種族がいる場合の差別化）
- **特定キャラ/場所の名前**（重要な固有名詞）
- **命名規則の方向性**

**選択肢は必ず番号付きで提示する:**

```
この世界の主要言語ベースを選んでください:
1. ラテン系（流麗で優雅な響き）
2. ゲルマン系（硬質で力強い響き）
3. ケルト系（神秘的で独特な響き）
4. 混合（地域・種族で使い分け）
5. その他（自由入力）
```

---

## 注意点

- 世界観との一貫性を最優先
- 読者が発音・記憶しやすい名前を心がける
- 同じ音で始まる名前を避ける（混乱防止）
- 既存の有名作品との被りに注意
- 重要キャラは意味のある名前、モブは簡素でよい
