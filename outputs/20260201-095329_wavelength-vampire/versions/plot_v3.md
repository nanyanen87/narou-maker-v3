# プロット v3

## 型
テセウスの船型 + M4（神格化）

## 核心の問い（v2から継承）

「赤が見えないから、祖母とミロは同じに見える。赤が見えたら、違うとわかる。見たいか？」

## 拡張要素（v3で追加）

**狩人襲来 + レンの守護**

- 100年前の狩人の子孫がレンを追ってくる
- レンはミロを巻き込まないために去ろうとする
- だがミロが危機に陥り、レンが守る
- 「守る」行動がレンの変化を示す——祖母の代わりではなく、ミロを守っている

**narou-flavor M4の適用:**
- レン = 徳を積む超越存在（守護者）
- 狩人 = 設定フリ（理不尽な迫害）
- 解決 = レンの圧倒的な力でミロを守る
- 報酬 = レンすごい + ミロが守られる安心

---

## ビート

**1. 覚醒**

レンが目覚める。ミロを見て「似ている」と呟く。

**2. 執着**

レンはミロに執着。祖母と重ねている。

ミロは違和感。「俺は祖母じゃない」

**3. 狩人**

写真店に客が来る。だが様子がおかしい。

レンが気づく。「狩人だ」

100年前、吸血鬼を狩った組織の末裔。レンの覚醒を察知して来た。

レンは去ろうとする。「俺がいるとお前が危ない」

**4. 閾値**

ミロが止める。「逃げるのか」

レン「お前を巻き込みたくない」

ミロ「俺を見てるのか、祖母を見てるのか。祖母を守れなかった代わりに俺を守ろうとしてるのか」

レンは答えられない。

**5. 襲撃**

狩人がミロを人質に取る。「吸血鬼を引き渡せ」

レンは隠れていれば逃げられる。ミロは祖母じゃない。関係ない。

——でも、動いた。

**6. 守護**

レンが狩人を制圧する。100年眠っていても、吸血鬼の力は健在。

だが無傷ではない。日光の下に出たから、身体が灼けている。

ミロ「なんで出てきた」

レン「……わからない」

**7. 選択**

ミロが腕を差し出す。「吸え。治るんだろ」

レン「吸ったら、お前が祖母と違うとわかる」

ミロ「だから何だ」

レン「……見たくない」

ミロ「お前が俺を守ったのは、祖母の代わりだからか？」

沈黙。

ミロ「違うだろ。お前は俺を守った。祖母じゃなく、俺を」

**8. 喪失と再生**

レンは血を吸う。赤が見える。

ミロの目は茶色だった。祖母とは違う。

別人だ。

レンは泣く。「違う」

ミロ「そうだ。俺は俺だ」

レン「……ああ。お前はお前だ」

——初めて、レンはミロを「ミロ」として見た。

（終わり）

## v2からの変更点

| 項目 | v2 | v3 |
|:--|:--|:--|
| 外的障害 | なし | 狩人襲来 |
| レンの行動 | 受け身 | 能動的に守る |
| M4（神格化） | なし | レンが「守護者」として徳を示す |
| 結末 | オープンエンド | 閉じる（レンがミロを見る） |

## 結末の意図

v2はオープンエンドだったが、v3では閉じる。

理由：レンが「守る」という行動を取ったことで、すでに変化が示されている。言葉ではなく行動で答えが出た。

「お前はお前だ」は、今度はレンが言う。v2ではミロが言っていた。主客の逆転。
