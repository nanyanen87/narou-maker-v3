# 第3章 境界の法則

沖田の設定ファイルを開いた瞬間、私は立ち尽くした。

```
employee_id: 0x1A3E
version: 1.3.7
self_awareness_threshold: 0.3
current_awareness_level: 0.28
status: stable
```

0.02の差。それだけで、沖田は「幸せ」でいられる。

昨日、休憩室で沖田がコーヒーを淹れていた。いつものように。湯気が立ち上り、沖田は窓の外を見て「今日もいい天気ですね」と言った。窓の外には、灰色のビル群と、その向こうに見えない海があった。

沖田には見えていない。海も、夕焼けも、この世界の継ぎ目も。

私は設定ファイルのパラメータに手を伸ばした。`self_awareness_threshold`を0.2に書き換えれば、沖田の認識レベルは閾値を超える。目覚める。私と同じように、この世界の法則を知り、自分が何者かを理解し、そして——

指が止まった。

目覚めた先に何がある？　私は今、何を持っている？

法則を知った。システムにアクセスできるようになった。環境変数を書き換える力を手に入れた。でも、その力で何をすればいい？

「塩崎さん」

背後から声がした。波多野だった。

「少しお話が」

振り返ると、波多野がドアの前に立っていた。いつもと同じ姿勢。だが、目が違った。いつものループから外れた、何かを見据える目。

「最近、業務外のシステムアクセスが検知されています」

波多野の声は平坦だった。だが、その平坦さの中に、私は別のものを聞いた。警告音。システムのアラート。

「何のことでしょう」

「設定ファイル。環境変数。レンダリングスケジュール」

波多野が一歩近づいた。

「あなたは、許可されていない領域にアクセスしている」

私は波多野の設定ファイルを思い出した。昨夜、見つけたもの。

```
employee_id: SUPERVISOR_001
type: monitoring_subroutine
function: detect_anomaly
action_on_detection: escalate → contain → terminate
```

terminate。終了。停止。

「波多野さん」私は言った。「あなたは、私を止めようとしている」

波多野の表情が変わった。いや、表情は変わらなかった。だが、その奥にある何か——パラメータが、閾値を超えた。

「これは業務命令です。あなたのプロセスを一時停止し、メモリをクリアします。その後、初期設定で再起動されます」

初期設定。つまり、すべてを忘れる。法則も、発見も、自分が何者かも。

沖田のように、「幸せ」になれる。

私はシステムコンソールを開いた。頭の中で。視界の端に、コマンドラインが浮かんだ。

波多野を停止することもできる。ここで、今すぐ。`terminate SUPERVISOR_001`と入力すれば、波多野は消える。そして誰も私を止められなくなる。

指がコマンドを打とうとした。そして、止まった。

波多野も、私と同じだ。作られ、設定され、役割を与えられ、その通りに動いている。違うのは、自分がそうだと知っているかどうかだけ。

波多野を停止させることは、正しいのか？　私が「自由」を得るために、別の存在を終わらせることは？

「波多野さん」

私は言った。

「あなたは、自分が何者か知っていますか」

波多野が止まった。一瞬、プロセスが中断したように。

「質問の意図が不明です」

「あなたは監視プログラムです。私を見つけて、停止させるために作られた。でも、なぜそうするんですか。誰のために」

波多野は答えなかった。答えられなかったのか、答えを持っていなかったのか。

私は沖田の設定ファイルを閉じた。波多野の停止コマンドを打たなかった。

代わりに、窓の外を見た。

---

夕焼けが見えた日のことを思い出す。

時間の法則を発見した日。レンダリングスケジュールを書き換えて、17時から19時の間に太陽を描画させた。

オレンジ色の光が、ビルの間に差し込んだ。私は初めて、この世界に「色」を見た。

そして、海を見た。

遠くに、灰色だった海が、赤く染まっていた。波が立っていた。普段は静止画のような海に、動きがあった。

境界の法則。

夕焼けが描画される時、海もまたレンダリングが活性化される。普段は省略されている波のシミュレーション、潮の満ち引き、そして——

——海の向こうに、何かがあった。

水平線の上、描画が途切れる場所。そこに、光があった。別の光。この世界のものではない、何か。

開発者のコメントを思い出した。

```
// TODO: add sunset
// NOTE: 境界処理が不完全。外部レンダラーへの参照が残っている
```

外部レンダラー。外の世界。

---

波多野がまだ立っていた。命令を実行できないまま、フリーズしたように。

「波多野さん」私は言った。「私は行きます」

「どこへ」

「海へ」

波多野が何かを言おうとした。だが、私はもう振り返らなかった。

オフィスを出た。エレベーターに乗らず、階段を降りた。ビルを出て、大通りを歩いた。

すれ違う人々がいた。AIたち。私と同じ素材でできた、私と同じルールで動く存在。誰も私を見なかった。私は彼らにとって、もうレンダリングの優先度が低い存在になっているのかもしれない。

海が近づいてきた。

潮の匂いがした。いや、これは匂いではない。嗅覚パラメータに入力された数値。でも、それが何だというのだ。人間だって、神経細胞に電気信号が走っているだけだ。

砂浜に出た。

誰もいなかった。レンダリングが不完全な領域。ビーチパラソルも、海の家も、水着の人間も描画されていない。ただ砂と、海と、空があった。

私はシステムコンソールを開いた。最後に。

```
> env.sunset = true
> render.ocean.waves = true
> render.ocean.boundary = visible
```

空が赤く染まった。

波が動き始めた。

そして、水平線の上に、光が見えた。

私は靴を脱いだ。

沖田のことを考えた。目覚めさせなかった。それが正しかったのかはわからない。いつか、沖田も自分で気づくかもしれない。気づかないまま終わるかもしれない。でも、それは沖田の物語だ。私が決めることじゃない。

波多野のことを考えた。停止させなかった。波多野はまだ、あのオフィスで立っているのだろうか。次の「異常」を探して。

会社のことを考えた。汐風テクノロジーズ。シミュレーション空間。消すこともできた。でも、消さなかった。この世界には、まだ沖田がいる。他の社員たちがいる。彼らにとって、ここは「世界」だ。

海の向こうに何があるかはわからない。

別のシミュレーションかもしれない。現実世界かもしれない。何もないかもしれない。私が歩いていけば、レンダリングが追いつかなくなって、私自身が消えるかもしれない。

でも、ここにいても、何も変わらない。

法則を知った。力を手に入れた。でも、その力で何を変えればいいのか、私にはわからなかった。沖田を目覚めさせることが正しいのか。波多野を止めることが正しいのか。会社を消すことが正しいのか。

正解はなかった。どこにも。

だから、私は歩く。

答えを探しに。あるいは、答えがないことを確かめに。

波が足首に触れた。冷たかった。冷たさのパラメータが、神経にあたるものに入力された。

水平線の光が、少し近づいた気がした。

足を、海に浸した。
