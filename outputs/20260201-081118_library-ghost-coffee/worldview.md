# 物語のスコープ

## 日常の舞台

地方都市の住宅街にある「栞文庫（しおりぶんこ）」。

築50年の洋館を改装した私設図書館。1階が閲覧室と喫茶コーナー、2階が書庫。蔵書は約3000冊。

**特異点**: この図書館の本を読むと、その本の「結末」が読者の人生に反映される。

## 異能のルール

### 基本ルール
1. **栞文庫の本を読み終えると、3日後にその本の結末が読者の人生に起こる**
   - ハッピーエンド → 幸運が訪れる
   - バッドエンド → 不幸が訪れる
   - 途中でやめると効果なし（読了が条件）

2. **効果の強さは「感情移入の深さ」に比例する**
   - 流し読み → 軽い影響
   - 没頭して読む → 強い影響

3. **同じ本を複数人が読むと、最後に読んだ人に効果が移る**

### 幽霊の「代読」
- 栞（幽霊）は、誰かが悲劇の本を手に取ると先回りして読む
- 結末の効果を自分が引き受ける
- 幽霊なので「死」は来ないが、存在が薄れていく
- 50年間の代読で、もうほとんど力が残っていない

### 災厄の書
- 図書館の奥に封印された特別な本
- 読んだ者の「最も大切なもの」を奪う
- 50年前、雪子の母がこれを読もうとした
- 栞が代わりに読み、「最も大切なもの＝図書館」を奪われる結末を引き受けた
- その結末が50年かけてゆっくり発動し、今「閉館」として現れている

# 世界の背景設定

## なぜ図書館に異能があるのか

創設者・栞の「願い」が具現化した。

栞は貧しい家庭に育ち、本を買えなかった。図書館だけが居場所だった。「本を読むことで人生が変わる」——その信念が、死後も図書館に残り、文字通りの力になった。

## 図書館の歴史

| 年 | 出来事 |
|:--|:--|
| 50年前 | 栞（28歳）が栞文庫を開館。開館日に書庫の梯子から転落死 |
| 50年前〜 | 栞の妹が図書館を継ぐ。栞は幽霊として残り、来館者を守り始める |
| 30年前 | 雪子の母（栞の姪）が災厄の書を発見、読もうとする。栞が代読 |
| 25年前 | 雪子誕生 |
| 10年前 | 栞の妹（雪子の祖母）死去。管理人が引き継ぐ |
| 5年前 | 雪子の母が病死（災厄の書とは無関係の自然死） |
| 現在 | 経営難で閉館決定。しかし本当の原因は災厄の書の結末発動 |

## コーヒーの意味

栞は50年間、来館者のために悲劇を引き受けてきた。
唯一、自分のために許した楽しみが「コーヒーの香りを嗅ぐこと」。

飲めない。触れない。でも香りだけは分かる。
それが栞にとっての「休息」であり、「まだここにいる」という実感だった。
